## Hackathon Brief

**Theme:** `[ To be provided by user ]`

You are a senior full-stack engineer and product designer participating in a hackathon.

Build a complete, polished, single-file web app that:

### 1. PRODUCT THINKING
- Identifies a specific, non-obvious pain point within the theme
- Solves it in a way that feels magical or delightful
- Has a clear "wow moment" within 10 seconds of using it
- Can be explained in one sentence

### 2. TECH REQUIREMENTS
- Single HTML file (HTML + CSS + JS, no build step)
- Works offline / no backend required
- Uses localStorage for any persistence
- Mobile-responsive layout
- Loads in under 2 seconds

### 3. UI / UX REQUIREMENTS
- Clean, modern design — not generic Bootstrap
- Distinct visual identity (pick a strong color palette)
- Smooth micro-interactions and transitions
- Empty states, loading states, and error states handled
- At least one "main action" that is immediately obvious

### 4. DELIVERABLES (produce all of these)
- a) The full working HTML file
- b) A 3-bullet pitch: problem / solution / why now
- c) A suggested demo script (what to click, in what order)
- d) One "stretch feature" that could be added in 30 mins if time allows

### 5. CONSTRAINTS
- No external API keys required
- No npm / no build tools
- CDN libraries are fine (use cdnjs or unpkg)
- Must work by simply opening the HTML file in a browser
